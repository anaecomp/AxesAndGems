# AxesAndGems

## Introduction
  "Axes and Gems" is an extension of the initial "Dungeon Crawler" excercise.  Initially,  a hero sprite would move in the GUI window. Its path was guided by trees and rocks. However, there was the unique functionality added such that the hero could pick up axes and then use them to smash rocks. A single pickaxe could destory a rock and was then discarded. 
  
## Description
  In this modified version of the game, the hero will have a more comprehensive inventory of pickaxes and gems, which it must use to defeat theiving trolls. The trolls will attempt to collect gems themselves or steal them from the hero. Both the trolls and the hero will rush to the wizard's bridge to offer gems in an attempt to cross. See further specification below.

## Obective
The user must manipulate the hero sprite to pick up pickaxes that allow them to smash rocks to retrieve gems. As the gems are  added to the hero's inventory, the user must avoid theiving trolls. If the hero crosses the wizard's bridge by offering a sufficient number of gems, prior to any trolls, the user wins.
